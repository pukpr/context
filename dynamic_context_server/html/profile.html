The fine terrain profiles can be modeled effectively by Markov and semi-Markov processes.

Markov processes such as Ornstein-Uhlenbeck model rough random terrain, while semi-Markov processes capture pseudo-random features with subtle periodicities.
For a profile, the model is Monte Carlo generated from the Markov or semi-Markov sequence.
For a PSD, the plot includes:

  • a Markov or semi-Markov stochastic representation
  • a windowed FFT of the Monte Carlo profile.
  • the FFT of the data.

High-resolution data from CSIR is stored in the database (see the sets labelled GEROTEK and Rough Track) as well as a few other test sets.

The FFT can show quite a bit of noise, so use the moving average filter to reduce the noisy values so that the underlying spectrum is clearly delineated. The graph can be stretched and then filtered to reveal the signal.
