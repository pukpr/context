The fine terrain profiles can be modeled effectively by Markov and semi-Markov processes.

Both CSIR data and a model fit will be plotted.
For a profile, the model is Monte Carlo generated.
For a PSD, the model includes:

  • a semi-Markov stochastic representation
  • a windowed FFT of the Monte Carlo profile.

The FFT can show quite a bit of noise, so use the moving average filter to reduce the noisy values so that the underlying spectrum is clearly delineated.
